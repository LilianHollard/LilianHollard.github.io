Application to textures

The main idea is to use a method that blurs an image using vector fields.

GLSL 
vec4 blur(sampler2D img, vec2 uv, vec2 resolution, vec2 direction){
	vec4 res = vec4(0.0);
	vec2 off1 = float * direction //direction corresponding to vector field
	color += texture(img, uv);
	color += texture(img, uv + off1/resolution, [...]);		
	//play with uv (which is gl_FragCoord.xy / iResolution.xy) and screen resolution
	//to blur the texture
	color += texture(img, uv - off1/resolution, [...]);		
	return color;
}

We can now call this method via our vector field method already present in the fragmentShader in order to return a color corresponding to a blur directed by an existing vector in the F function created by the vector field.
